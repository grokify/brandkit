## Versioning

BrandKit follows [Semantic Versioning](https://semver.org/):

- **MAJOR** (1.0.0) — Breaking changes
- **MINOR** (0.x.0) — New features, backwards compatible
- **PATCH** (0.0.x) — Bug fixes
