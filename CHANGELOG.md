## [0.2.0] - 2026-01-25

### Added

- Go library API for embedded icon retrieval: `GetIcon`, `GetIconWhite`, `GetIconColor`, `GetIconOrig`, `ListIcons`, `IconExists`, `NormalizeIconName` ([`2c95642`](https://github.com/grokify/brandkit/commit/2c95642))
- Go library API for icon processing: `ProcessWhite`, `ProcessColor` for programmatic SVG processing ([`ea6e409`](https://github.com/grokify/brandkit/commit/ea6e409))
- Go library API for recursive verification: `svg.ListSVGFilesRecursive`, `verify.DirectoryRecursive` ([`ab5324f`](https://github.com/grokify/brandkit/commit/ab5324f))
- CLI command `brandkit color` for creating centered color icons preserving original colors ([`8a8eecc`](https://github.com/grokify/brandkit/commit/8a8eecc))
- CLI command `brandkit verify-all` for recursive pure vector verification (CI-friendly) ([`8a8eecc`](https://github.com/grokify/brandkit/commit/8a8eecc))
- 17 new brand icons: bolt, bootstrap, dart, flutter, go, javascript, kotlin, lovable, openapi, postgresql, postman, python, react, replit, spring, v0, windsurf (total: 52 brands) ([`205a287`](https://github.com/grokify/brandkit/commit/205a287))
- 42 new `icon_color.svg` files for all brands with transparent backgrounds ([`bcf0c80`](https://github.com/grokify/brandkit/commit/bcf0c80))
- GitHub Actions workflow `verify.yaml` for automated icon validation on PR/push ([`0358ff9`](https://github.com/grokify/brandkit/commit/0358ff9))

### Changed

- Renamed `gcp` brand directory to `google-gcp` for clarity and consistency ([`78afca3`](https://github.com/grokify/brandkit/commit/78afca3))

## [0.1.0] - 2026-01-24

### Added

- CLI command `brandkit white` for one-step white icon generation with background removal, color conversion, centering, and verification
- CLI command `brandkit convert` for SVG color conversion with support for hex, shorthand, and named colors
- CLI command `brandkit process` for full SVG processing pipeline (convert, center, verify)
- CLI command `brandkit analyze` for SVG geometry analysis (centering, padding, viewBox optimization)
- CLI command `brandkit verify` for pure vector validation (detects embedded base64, data URIs, binary references)
- Public `svg` package with `BoundingBox`, `ViewBox`, `ParsePath`, `CalculatePathBounds`, and file utilities
- Public `svg/convert` package for programmatic SVG color conversion with background removal and mask preservation
- Public `svg/analyze` package for programmatic SVG geometry analysis
- Public `svg/verify` package for programmatic pure vector validation
- Brand asset library with 36 brand icon directories and standardized SVG variants
